В файлах APP.config, в  ContractCurs и DAL необходимо заменить строку подключения на свой собственный SQL SERVER;

Запустить проект, на основе классов создастся модель базы данных;
Закрыть программу;
Запустить файл инициализации стартовых данных в SQLinit/InitialDB.sql;

Проект готов к использованию;

При необходимости можно снова уничтожить базу с помощью SQLinit/DBDROP.sql;
